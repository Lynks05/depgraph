setDataModel()

Set a data model to the edit object. The data model must be an object with one or all of the following properties arrays : - links - words - chunks Each of these arrays (except links which is an array containing exactly 1 element) contains objects that define labels model as following : - name : name of the label - values : (optional) array of possible values for label - value-restrict : (optional, default false) boolean indicating if the user input is restricted to defined possible values - onchange : (optional) user callback called after user submitted new value to this label field
